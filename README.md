#### **How to Run**  
1. Clone the repo:  
   ```bash
   git clone https://github.com/mehrdadMP/profile_project.git
   ```  
2. Install dependencies:  
   ```bash
   flutter pub get
   ```  
3. Run on emulator/device:  
   ```bash
   flutter run
   ```
